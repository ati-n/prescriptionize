# 💊 Prescriptionize

An app to help doctors keep a list of medicines they yet to prescribe to patients. (Hungarian language only)

## 🎯 Features

- Web API using ASP.NET (w/ .NET 6)
- Controller with CRUD methods
- Postgres server
